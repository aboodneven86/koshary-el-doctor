# 🍽️ موقع كشري الدكتور

موقع احترافي لمحل كشري الدكتور مع نظام طلبات أون لاين وتتبع الأوردر

## ✨ الميزات

- 🏠 صفحة رئيسية احترافية
- 🍽️ قائمة كاملة بالأطباق والأسعار
- 📱 نظام طلب أون لاين
- 📍 خريطة وبيانات المحل
- ⏰ ساعات العمل (24 ساعة)
- 📞 تواصل مباشر (واتساب - تليفون)
- 🔔 إشعارات الطلبات على الواتساب
- 📊 تتبع الأوردر في الوقت الفعلي
- 👥 قاعدة بيانات العملاء
- 🎨 تصميم عصري وجميل بالألوان الأخضر والأصفر

## 🛠️ التقنيات المستخدمة

- **Frontend**: React.js
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **WhatsApp API**: Twilio / Baileys
- **Hosting**: Vercel + Railway
